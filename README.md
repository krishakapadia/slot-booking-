# Smart Slot Booking System

A full-stack slot booking application inspired by real-world scheduling and ticketing systems.  
The platform allows administrators to create time slots and users to book available slots with real-time UI updates.

This project is being developed incrementally following industry best practices.

---

## Features

### User
- View available slots
- Book slots in real-time
- Slot availability updates instantly

### Admin
- Create new slots
- Manage slot availability
